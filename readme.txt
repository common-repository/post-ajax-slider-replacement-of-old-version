=== Post Ajax Slider (replacement) ===
Tags: Post, ajax, slider, postajaxslider, animation, shift, blog
Requires at least: 3.2
Tested up to: 3.1
Stable tag: 1.0

This plugin is a replacement of my last plugin (post-ajax-slider). 

== Description ==

This plugin is intended to ease the user to have a very updateable post while writer of the blogs currently writing (need to be posted first) the post. With this plugin, you can see your post updated in a realtime (without refreshing your current page). This plugin will provide you a better look of your current home page, so the user can easily see what your new post was.
